hslogon.C File Reference

Sets up HaSpect environment when included in .rootrc file.

Usage

Use flags when starting root to load needed functions

  • &ndash;hsfit : Classes for sWeight fits etc
  • &ndash;hssel : Classes for hselector
  • &ndash;hsdata : Classes for hsdata
  • &ndash;hsfinal=MyFinalState : Classes for hsfinalstate where
    MyFinalState is the users own derived class

e.g.

root -l --hsfit FitHSSimple.C

to start FitHSSimple.C macro which uses sWeights to fit data.

root --cleanall

to remove all .so .d and .pcm files

root RunSomeScript.C --proof=N

to initialise a proof session with N workers

root --MyClass.C

to load your own class from current directory, or a class from HSANA or HSUSER
